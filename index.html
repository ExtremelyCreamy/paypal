Paypal express checkout by Martin Maly

PayPal Express Checkout implementation in PHP

Implementing PayPal's full API isn't a piece of cake. There are some libraries around the web, but the most of them are paid, closed or too complex. If you wanna implement simple and single-item payments (i.e. "pay for service" or "pay for source code"), you can use this free and open library.

Dependencies

This library was written as a part of my framework, but it can be used standalone too, I hope. ;)
